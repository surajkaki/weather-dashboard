# Live Weather Dashboard

Real-time weather dashboard built with Streamlit and Open-Meteo API.

## Features
- Live current conditions for any city worldwide
- 5-day hourly forecast — temperature, humidity, wind
- Automatic anomaly detection
- Auto-refreshes every 30 or 60 minutes
- No API key or sign-up required

## Run Locally
pip install -r requirements.txt
streamlit run weather_dashboard.py

## Data Source
[Open-Meteo](https://open-meteo.com) — free and open, no authentication needed.
